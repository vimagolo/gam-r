# Proyecto **GAM-R**

**GAM-R** es un proyecto académico enfocado en el desarrollo y maquetación de una página web estática, implementando **buenas prácticas de estructura, diseño y control de versiones** mediante **Git y GitHub**.  
El proyecto fue desplegado en **Netlify** como parte de una actividad orientada al aprendizaje de los fundamentos del desarrollo web.

---

## Objetivo del proyecto

El propósito principal de **GAM-R** es aplicar los conocimientos adquiridos sobre **maquetación web**, **estructuración semántica en HTML** y **estilización con CSS**, garantizando un resultado visualmente coherente y técnicamente correcto.  
Además, el proyecto fomenta el uso de **Git y GitHub** como herramientas esenciales para la **gestión de versiones** y la **colaboración en entornos de desarrollo**.

---

## Tecnologías utilizadas

| Tecnología | Descripción |
|-------------|-------------|
| **HTML5** | Estructura y contenido del sitio web. |
| **CSS3** | Diseño visual, estilos y maquetación. |
| **Git & GitHub** | Control de versiones y gestión del repositorio. |
| **Netlify** | Despliegue y alojamiento del proyecto. |
